# Online-Banking-System
A secure web-based Online Banking System with Admin and Customer modules developed using web technologies and database connectivity.

## Features
- Customer Registration & Login
- Admin Login
- Account Approval
- Balance Enquiry
- Money Transfer
- Beneficiary Management
- Transaction History

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
